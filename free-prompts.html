🏡 Listing Descriptions (5 Prompts)

Prompt 1: Standard Listing

Write a compelling MLS listing description for a [X bed / X bath] home in [neighborhood, city]. Key features: [list 3-5 features]. Target buyer: [profile]. Under 200 words. Lead with lifestyle, highlight top 2 features, close with soft CTA.

Prompt 2: Luxury Listing

Write a premium listing description for a [sq ft] luxury home in [neighborhood]. Features: [list]. Buyer profile: affluent professional or family. Tone: sophisticated and aspirational. Under 220 words. Open with experience, not features.

Prompt 3: Starter Home

Write a listing description for an entry-level [X/X] home in [city]. Key selling points: [list]. Target buyer: first-time homeowner. Tone: warm and encouraging. Emphasize value, location, and potential. Under 180 words.

Prompt 4: Condo/Townhome

Write an MLS listing description for a [X/X] condo in [building/neighborhood]. Features: [list]. Target buyer: urban professional. Highlight: low maintenance, building amenities, walkability. Under 180 words.

Prompt 5: Fixer-Upper

Write an honest, compelling listing description for a fixer-upper [X/X] in [neighborhood]. Bones are good but needs updating. Target buyer: investor or customizer. Frame work needed as opportunity. Under 180 words.

📧 Follow-Up Emails (5 Prompts)

Prompt 6: Post-Showing Follow-Up

Write a follow-up email to a buyer after showing [property]. They seemed interested but non-committal. Warm, not pushy. Include one market insight. Ask one low-pressure timeline question. Under 100 words.

Prompt 7: Quiet Buyer Re-Engagement

A buyer went quiet after showing 2 weeks ago. Write a re-engagement email that acknowledges silence without guilt, adds market value, and ends with an easy yes/no question. Under 120 words.

Prompt 8: New Listing Alert

Write an email to a buyer client alerting them to a new listing matching their criteria: [criteria]. Create urgency without alarmism. Include: key highlights, why it fits, clear next step. Under 130 words.

Prompt 9: Post-Closing Thank You

Write a post-closing thank you email to [buyer/seller]. Heartfelt, genuine, not transactional. Acknowledge what we accomplished together. Soft referral ask at end. Under 120 words.

Prompt 10: Cold Lead Re-Activation

A lead went cold 3 months ago. Write a re-activation email that doesn't reference the gap, adds immediate value (market update), and invites them back without pressure. Under 110 words.

📱 Social Media (5 Prompts)

Prompt 11: Market Update Post

Write a Facebook post for real estate agents in [city] sharing a local market update. Data: [describe conditions]. Educational, not alarming. Under 150 words. End with engagement question.

Prompt 12: Home Buying Tip

Write a LinkedIn post sharing one underrated tip for first-time homebuyers. Not obvious advice. Makes agents look knowledgeable. Under 120 words. Conversational. End with question.

Prompt 13: 30-Day Content Batch

I'm a real estate agent in [city]. Generate 30 social posts for next month. Mix: 10 market updates, 10 tips, 5 neighborhood spotlights, 5 motivational. Each under 150 words. No hard selling. End with question.

Prompt 14: Just Listed Post

Write an exciting "just listed" Facebook post for [property description]. Highlight: [top 3 features]. Tone: enthusiastic but not over-the-top. Include CTA for showings. Under 100 words.

Prompt 15: Client Success Story

Write a social post celebrating a recent client win: [brief story]. Warm and genuine. Make client the hero. Soft mention of your role. Under 130 words. No client names.

💬 Objection Handlers (5 Prompts)

Prompt 16: "I'll Wait Until Rates Drop"

A buyer says "I'll wait until rates come down." Write 3 professional responses that acknowledge concern, reframe with data (prices vs rates), and suggest path forward. Each under 80 words. Not pushy.

Prompt 17: "Your Commission Is Too High"

A seller questions your commission. Write 3 responses that acknowledge perspective, articulate value without defensiveness, and pivot to results. Each under 75 words. Confident.

Prompt 18: "My Home Is Worth More"

A seller believes home is worth $50k more than CMA. Write 3 empathetic, data-backed responses keeping listing intact. Each under 80 words. Acknowledge emotion, ground in reality.

Prompt 19: "I Want to Try FSBO First"

A seller wants to try FSBO before listing. Write 3 responses that respect decision, highlight FSBO risks, and leave door open. Not condescending. Under 80 words each.

Prompt 20: "Another Agent Offered More"

A seller says another agent quoted higher price. Write 3 responses differentiating your approach without badmouthing. Focus: pricing methodology, data, track record. Each under 80 words.

🤝 Consultations & Scripts (5 Prompts)

Prompt 21: Buyer Consultation Questions

Generate 12 discovery questions for first-time buyer consultation. Uncover: priorities, timeline, budget flexibility, deal-breakers, emotional drivers. Conversational, not interrogation-style. Organized by topic.

Prompt 22: Listing Appointment Prep

I have listing appointment with [couple/individual] in [X years]. Property: [description]. Give me: 5 rapport-building openers, 3 ways to present CMA diplomatically, 2 responses to "other agent quoted higher."

Prompt 23: Agent Bio

Write a professional agent bio for [name] in [city]. [X] years experience. Specializes in [niche]. 150 words. Story-driven, not credential-heavy. Third person. Human and trustworthy, not corporate.

Prompt 24: Expired Listing Script

Write a phone script for reaching out to an expired listing. Seller is likely frustrated. My angle: I know why it didn't sell and have a different strategy. Empathetic, solution-focused. Under 200 words. Include 2 questions.

Prompt 25: Price Reduction Recommendation

Write an email recommending a price reduction after 30 days on market. Data: [showing count, feedback, comps]. Empathetic, not alarming. Frame as proactive. Keep relationship strong. Under 150 words.
